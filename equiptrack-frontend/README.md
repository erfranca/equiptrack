# EquipTrack Frontend (React + Vite + Tailwind)

## Rodar local
```bash
npm install
npm run dev
```
- URL: http://localhost:5173
- Configure a API (se necessário) via `.env`:
```
VITE_API_URL=http://localhost:3001
```
Por padrão, já usa `http://localhost:3001`.

## Fluxo
- **Login** (admin@example.com / admin123)
- **Dashboard**: atalhos
- **Registrar**: Entrada/Saída, Instalação/Manutenção, Cliente, Tipo, Série, Acessórios/Defeitos
- **Busca**: filtros e listagem
- **Clientes**: cadastro e listagem
